# Customer Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to identify purchasing patterns and business insights using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI
- Jupyter Notebook

## Dataset
The dataset contains customer purchase details including age, gender, product category, purchase amount, and subscription status.

## Key Insights
- Clothing category generates the highest revenue
- Young adults contribute the highest revenue
- 73% customers are non-subscribers
